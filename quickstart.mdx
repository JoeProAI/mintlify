# Quickstart Guide

Get started with VitalMatrix in just a few minutes.

## Installation

```bash
# Clone the repository
git clone https://github.com/vitalmatrix/vitalmatrix.git
cd vitalmatrix

# Install dependencies
npm install

# Start the development server
npm run dev
```

Visit `http://localhost:3000` to see the application in action.

## Key Features to Try First

1. **NutriLens Camera**: Scan food items to get instant nutritional information
2. **Food Search**: Look up nutrition data for thousands of common foods
3. **Daily Log**: Track your meals and see a breakdown of your nutritional intake

## Configuration

Create a `.env.local` file with the following variables:

```
NEXT_PUBLIC_API_URL=https://api.vitalmatrix.dev
NEXT_PUBLIC_ENABLE_DEBUG=false
```
